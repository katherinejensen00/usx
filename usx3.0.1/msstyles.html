<ms> @style Types¶

Note

Numbered <ms> @style Attributes

As with USX <para> elements which include a numeric variable in the @style attribute, a number may be added to a milestone @style attribute to indicate a relative weighting or level.

style = style1 — The unnumbered version of a style attribute may be used when only one level of this milestone style exists within the project text. Numbers should always be included when more than one level of the milestone style exists within the project text.

qt#¶

@style:qt#
Use:Quotation speaker.
# denotes the level of nesting of the quotation being marked (i.e. a quote within a quote).
qt = qt1 (see notes on numbered @style attributes)
Valid In:Any valid <ms>

Attributes

Additional attributes for <ms> which can be used in this context.

@who:The speaker of the current quotation. 
<ms style="qt1-s" who="Paul"/>

ts¶

@style:ts
Use:Translator’s section.
For identifying a section (chunk) of text suitable for translating at one time.
Valid In:Any valid <ms>
